# Distro-hopping configs #
## i3  ##
> i3 config path is
> ~/.config/i3/config
## .i3status.conf  ##
> .i3status.conf path is
> ~/.i3status.conf
## xterm config ##
> xterm config path is 
> ~/.Xresourses
> xrdb -merge .Xresourses
command is required to apply changes
## X11 configs for monitors and keyboard##
> the path is 
> /etc/X11/xorg.conf.d
## neovim config ##
> the path is 
> ~/.config/nvim/init.vim
## zsh config ##
> the path is
> ~/.zprofile and ~/.zshrc
